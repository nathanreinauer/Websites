WHAT I DO

THE WEB
My very first introduction to coding was making websites in nothing but HTML for my (many) bands as a kid. Over the years I've continued to hone my skills and have created websites for small businesses, city governments, interior designers, and many others. I love playing with CSS and pushing it to its limits where others might use JavaScript for the same effect. That said, I'm quite comfortable with JavaScript and JQuery. And if you need something a little more powerful and dynamic, I can do ASP.NET as well.

SOFTWARE & PROGRAMMING
I fell in love with software development while studying at The Tech Academy in Portland, OR. My favorite language is Python since you can do so much with so little, but I also know C# and the .NET Framework for more advanced projects. If you need help with database creation/management, I'm very comfortable with SQL as well. I'm always learning more, and if a situation calls for another language, I'm a very quick learner.

MY TOOLS
A person is only as good as his or her tools, and I have a lot on my proverbial belt. I'm comfortable with many popular IDE's for web development and software design, and I regularly use several different database management tools. My experience is mainly Windows and Microsoft-based, but I know my way around a Mac, too. I also believe version control is essential, and I spend a lot of time working with Github so nothing is ever lost.
